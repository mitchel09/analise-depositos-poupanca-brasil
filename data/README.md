# Dados

Esta pasta contem os dados utilizados na analise.

## Arquivo: ipeadata_2017_2022.xls

Fonte: IPEA Data / Banco Central do Brasil (BACEN)
Documento de origem: Estatistica Bancaria Mensal/Global
Periodo: 2017 a 2022
Registros: 137 meso-regioes brasileiras
Variaveis: 9 colunas

## Como obter os dados

Os dados podem ser acessados diretamente pelo IPEA Data (http://www.ipeadata.gov.br/) ou pelo notebook no Google Colab:

https://colab.research.google.com/drive/1JcSxdFeFVnaa6f92QJJb-pvB9uVAJfwK?usp=sharing

Nota: O arquivo .xls nao esta versionado neste repositorio por conta do tamanho. Utilize o link acima para acessar a analise completa com os dados ja carregados.
